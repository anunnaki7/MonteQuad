**Monte KL** is a high-performance portfolio concept that merges the raw, unapologetic aesthetic of **Neo-Brutalism** with a fluid **Liquid UX** approach. This project is a deep dive into high-contrast design, bold typography, and surgical-grade code optimization.

[**🌐 Live Demo**](https://brutal-ui-liquid-ux.vercel.app/) • [**📂 Repository**](https://github.com/anunnaki7/brutal-ui-liquid-ux)

---

## 💎 Core Philosophy

* **🔲 Neo-Brutalist Aesthetic:** Heavy borders, bold monochrome palettes, and a "raw" design philosophy that prioritizes content and structure.
* **💧 Liquid Animations:** Leverages the `Intersection Observer API` for seamless "on-scroll" reveals and dynamic interactions without sacrificing performance.
* **🎯 Surgical Performance:** Optimized asset delivery and clean vanilla architecture to ensure top-tier Core Web Vitals.
* **📱 Responsive Precision:** A fully fluid layout engineered to scale perfectly across all breakpoints, from mobile screens to ultra-wide displays.

## 🚀 Installation & Local Development

To explore or modify this project locally:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/anunnaki7/brutal-ui-liquid-ux.git](https://github.com/anunnaki7/brutal-ui-liquid-ux.git)
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd brutal-ui-liquid-ux
    ```

3.  **Run the project:**
    Open `index.html` using a local server (e.g., Live Server in VS Code) to maintain all functionalities.
