# ⚡ Monte KL - Brutalist UI / Liquid UX

Portfolio sajt baziran na principima **neo-brutalizma** i **tečnog korisničkog iskustva**. Projekat je fokusiran na visoki kontrast, čistu tipografiju i glatke animacije.

## 🚀 Tehnologije
* **HTML5** - Semantička struktura
* **CSS3** - Custom Properties (variables), Flexbox, Grid, Keyframe animacije
* **JavaScript** - Intersection Observer API za "on-scroll" animacije
* **Vercel** - Deployment i hosting

## ✨ Ključne karakteristike
- **Responsive Design:** Potpuno optimizovano za mobilne uređaje (bez horizontalnog scroll-a).
- **Brutalist Aesthetic:** Jaki borderi, visok kontrast i beskompromisan dizajn.
- **Liquid Animations:** Elementi se pojavljuju dok skrolujete, stvarajući dinamičan osećaj.
- **Clean Code:** Organizovan CSS i optimizovana struktura fajlova.
